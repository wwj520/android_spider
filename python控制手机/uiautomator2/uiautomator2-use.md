### 基本使用：
#### 安装：
- pip 安装
    pip install uiautomator2  # 这个包是用于控制手机
    pip install pillow  # 这个包用于截图
    pip install -U weditor
    
- 设备安装ATX
    python3 -m uiautomator2 init

- weditor调试工具
    命令窗口输入：python3 -m weditor
